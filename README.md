# 📚 Enciclopedia del Desarrollador

> Portfolio personal con estética de enciclopedia clásica

## 🎯 Concepto

Este portafolio no es solo una lista de proyectos, es un compendio de conocimiento construido a través del tiempo. Cada proyecto es presentado como un artículo enciclopédico detallado.

## ✨ Características

- **Portada Elegante**: Diseño tipo libro clásico con animaciones sutiles
- **Índice General**: Organizado por categorías (A-E)
- **Fichas Enciclopédicas**: Cada proyecto con estructura académica:
  - Resumen
  - Contexto
  - Metodología (Arquitectura, Tecnologías, Patrones)
  - Procedimiento
  - Resultados
  - Aprendizajes
  - Referencias
- **Glosario Técnico**: Conceptos explicados con palabras propias
- **Línea del Tiempo**: Cronología del aprendizaje

## 🛠️ Tecnologías

- **React 18** - Framework UI
- **Vite** - Build tool
- **React Router** - Navegación
- **Tailwind CSS** - Estilos
- **Framer Motion** - Animaciones
- **Merriweather & Inter** - Tipografía



## 📁 Estructura del Proyecto

```
src/
├── pages/           # Páginas principales
│   ├── Cover.jsx    # Portada
│   ├── Index.jsx    # Índice general
│   ├── ProjectDetail.jsx
│   ├── Concepts.jsx
│   └── Timeline.jsx
├── data/            # Datos del contenido
│   ├── projects.js
│   ├── concepts.js
│   └── timeline.js
├── App.jsx          # Router principal
└── index.css        # Estilos globales
```

## 🎨 Tema Visual

- **Fondo**: Paper (#FAF9F6) - Tono papel antiguo
- **Texto**: Ink (#1A1A1A) - Negro suave
- **Acento**: Sepia (#704214) - Detalles cálidos
- **Énfasis**: Blue Ink (#1E3A5F) - Títulos principales
- **Tipografía Serif**: Merriweather (títulos)
- **Tipografía Sans**: Inter (cuerpo)
